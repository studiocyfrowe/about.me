# Dominik Hofman – Inżynier AI, Programista Backend

Tworzę systemy oparte na AI, które automatyzują rzeczywiste przepływy pracy z wykorzystaniem LLM, agentów i usług backendowych.

---

## O Mnie

Rozpocząłem karierę w projektowaniu graficznym i web design. Pod wpływem zainteresowania nowymi technologiami preszedłem do działu wsparcia IT, aby lepiej zrozumieć procesy zachodzący w obszarze informatyki w obrębie dużych przedsiębiorstw. Jestem pragmatykiem, co oznacza, że pragnienie studiowania zagadnień związanych ze Sztuczną Inteligencją zaspokajałem praktyką - projektowaniem i programowaniem aplikacji w językach C# oraz Python. Wiedzę o rozwiązywaniu rzeczywistych problemów informatycznych użytkowników starałem się odwzorować w moim flagowym projekcie - SolveDesk.

Moim celem zawodowym jest przejście do inżynierii AI, budując praktyczne systemy, które łączą:

* programowanie backendu
* modele AI (LLM)
* automatyzację przepływów pracy

Kieruję się ku projektowaniu rozwiązań, które redukują pracę ręczną i poprawiają wydajność w środowiskach wsparcia i operacyjnych.
